# 🤖 MyMate Pro v3 인수인계 문서 (Handover Document)

본 문서는 일반 사용자도 쉽게 사용할 수 있는 초개인화 RPA 도구인 **MyMate Pro**의 현재 구현된 기능과 기술 사양을 정리한 문서입니다.

## 1. 프로젝트 개요
- **제품명:** MyMate Pro (Studio)
- **목표:** 코딩 지식이 없는 일반인이 마우스와 키보드 동작을 자유롭게 자동화하고, 이미지/색상 인식을 통해 지능적인 업무 처리를 돕는 도구.
- **주요 특징:** 세로형 노드 기반 빌더, 복합 조건 인식(좌표+이미지+색상), 실시간 녹화 통합.

---

## 2. 기술 스택 (Tech Stack)
- **Framework:** Electron + Next.js (Nextron)
- **Language:** TypeScript
- **Styling:** Tailwind CSS, Lucide React (Icons)
- **Automation Engine:** 
  - `@nut-tree-fork/nut-js` (마우스/키보드 제어)
  - `uiohook-napi` (전역 입력 녹화)
  - `screenshot-desktop` & `jimp` (화면 캡처 및 이미지 분석)

---

## 3. 핵심 기능 명세

### 🖱️ 마우스 제어 (Mouse Control)
- **다양한 클릭 지원:** 좌클릭, 우클릭, 휠 클릭, 더블 클릭.
- **드래그 앤 드롭:** 시작 좌표와 끝 좌표를 지정하여 요소 이동 가능.
- **휠 스크롤:** 위/아래 방향과 양(Amount) 조절 가능.
- **마우스 연타:** 특정 지점을 설정한 횟수와 간격(ms)으로 무한/연속 클릭.

### ⌨️ 키보드 입력 (Keyboard Input)
- **텍스트 자동 입력:** 미리 설정된 긴 문장을 타이핑 모드로 입력.
- **단축키 실행:** `Ctrl+C`, `Alt+Tab` 등 복합 단축키 완벽 지원.
- **키보드 연타:** 특정 키(Enter, Space 등)를 반복적으로 입력.

### 👁️ 지능형 인식 엔진 (Vision & Recognition)
- **좌표 지정:** 화면의 특정 위치(X, Y)를 직접 찍어서 기억.
- **이미지/UI 인식:** 화면상의 특정 버튼이나 아이콘을 캡처하여, 위치가 바뀌어도 찾아가서 클릭.
- **색상 인식:** 특정 좌표의 픽셀 색상(#HEX)을 추출하고, 실행 시 일치 여부 검증.
- **복합 조건 조합:** **한 블록 안에 [좌표 + 색상 + 이미지]를 모두 포함** 가능. (세 조건이 모두 맞을 때만 동작하도록 설계)

### 🔀 흐름 제어 (Flow Control)
- **시간 대기 (Wait):** 작업 사이사이에 밀리초(ms) 단위의 대기 시간 삽입.
- **조건부 분기 (IF):** 특정 이미지가 화면에 있을 때(TRUE)와 없을 때(FALSE)의 실행 경로를 다르게 설정 (n8n 스타일 트리 구조).

### 🔴 슈퍼 레코드 (Super Recorder)
- **실시간 행동 기록:** 사용자의 실제 마우스/키보드 동작을 실시간으로 감지하여 타임라인 블록으로 자동 변환.
- **수동 블록 통합:** 녹화된 블록 사이사이에 수동으로 만든 인식 블록이나 조건문을 자유롭게 배치 가능.

---

## 4. UI/UX 디자인 포인트
- **세로형 타임라인:** 작업 순서를 위에서 아래로 한눈에 파악할 수 있는 직관적인 레이아웃.
- **정보 투명성:** 블록을 클릭하지 않아도 **좌표, 텍스트 내용, 색상 코드**가 블록 위에 배지로 표시됨.
- **자동 숨김 모드:** 녹화나 타겟 지정(Picking) 시 프로그램 창이 자동으로 최소화되어 작업 방해를 최소화함.
- **실행 상태 시각화:** 현재 어떤 단계가 실행 중인지 **모래시계(스피너)**로 실시간 표시.

---

## 5. 주요 단축키 및 설정
- **비상 정지 (Panic Button):** `F12` (실행 중 즉시 중단 및 창 복구)
- **녹화 토글:** `F9` (앱을 열지 않고도 외부에서 녹화 시작/종료)
- **파일 관리:** 만든 매크로를 `.json` 형태로 저장하고 언제든 다시 불러오기 지원.

---

## 6. 향후 확장 가능성
- **OCR 글자 인식:** 이미지 대신 화면상의 글자를 읽어 조건문 작성.
- **엑셀 연동:** 엑셀 데이터를 읽어와 반복문(Loop) 처리.
- **랜덤 지연/좌표:** 사람처럼 보이기 위한 마우스 궤적 및 시간 오차 적용.
